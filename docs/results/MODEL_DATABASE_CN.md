## 关键见解与模式

### 1. **越小越好**（最重要的发现！）
对于在该数据集上从头训练的 CNN：
- SE-ResNet18 (0.8585) > SE-ResNet34 (0.8538) > SE-ResNet50 (0.8528)
- DenseNet-121 (0.8514) > DenseNet-169 (0.8430)
- EfficientNet-B0 (0.8492) > B2 (0.8472) > B4 (0.8428) > B7 (0.6670)
- ResNet-18 (0.8498) > ResNet-34 (0.8365)

**原因**：约 4,000 个训练样本不足以训练大型模型 -> 过拟合

### 2. **SE 块至关重要**
- SE-ResNet18 (0.8585) 与 ResNet-18 (0.8498) 相比 = **+8.7% 改进**
- Squeeze-and-Excitation 注意力对医学影像至关重要

### 3. 模型类型的训练策略
| 模型类型 | 最佳策略 | 典型 AUC |
|------------|---------------|-------------|
| 小 CNN（18-34 层） | 从头训练 | 0.85-0.86 |
| 大 CNN（50+ 层） | 从头训练（较差） | 0.85 |
| 大 Transformer | 冻结 -> 微调 | 0.84-0.85 |
| 小 Transformer | 不使用 | 0.54-0.65 |

### 4. 超参数影响
| 超参数 | 发现 |
|----------------|---------|
| 学习率 | 0.0005 > 0.001 对小模型 |
| 批次大小 | 8-12 最佳，更大 = 更差 |
| 早停耐心 | 15 > 10 用于处理方差 |
| 数据增强 | 必要（始终启用） |

### 5. 模型家族排名
1. **SE-ResNet**：0.8528-0.8585（最佳）#1
2. **DenseNet**：0.8430-0.8514
3. **标准 ResNet**：0.8349-0.8498
4. **EfficientNet**：0.6670-0.8492
5. **MobileNet**：0.8463-0.8483
6. **ConvNeXt**（冻结+微调）：0.8378-0.8540
7. **Inception**：0.8459
8. **UNet3D**：0.8024-0.8438
9. **ViT/Swin**（冻结）：0.7746-0.8144
10. **从头训练的大型模型**：失败

## 跳过失败的模型

### 完全训练崩溃（AUC < 0.70）
- **EfficientNet-B7**：0.6670（太大，OOM 问题）
- **ConvNeXt 从头训练**：0.6740（需要预训练）
- **ViT 从头训练**：0.6477（需要预训练）
- **ResNet-101**：~0.55（灾难性失败，太深）
- **Swin 从头训练**：0.5422（需要预训练）

### 失败原因
1. **对于约 4,000 个训练样本参数过多**
2. **Transformer 需要 ImageNet 预训练**（冻结骨干策略）
3. **3D 卷积**指数级增加参数数量
4. **非常深的网络中的梯度消失**（101+ 层）
